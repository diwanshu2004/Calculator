<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the Repository:</p>

```
git clone https://github.com/diwanshu2004/Calculator.git
```

<p>2. Install Dependencies:</p>

```
npm install
```

<p>3. Start Application:</p>

```
npm start
```
